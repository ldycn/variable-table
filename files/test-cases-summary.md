# 测试用例简述

## TC-01 表格展示

- 打开页面应显示空表格，有 5 列（索引/名称/数据类型/默认值/注释），索引列不可编辑，页面上有"添加行"和"删除行"两个按钮。

## TC-02 添加行

- 空表点添加 → 出现第 1 行，索引为 1，所有字段为空。
- 有数据时点添加 → 在末尾追加一行，索引 = 最大索引 + 1。
- 连续添加多行，索引依次递增，已有数据不受影响。

## TC-03 删除行

- 选中某行删除后，该行消失，剩余行索引自动重新排列。
- 删到一行不剩，表格变空，按钮仍在。
- 没选中行就点删除，不会误操作。
- 删除后再添加，索引继续递增。

## TC-04 编辑名称

- 单元格内容以input框显示。
- 输入唯一名称 → 保存成功。
- 清空名称 → 恢复原值并报错。
- 输入已有名称（不区分大小写）→ 报错"名称已存在"，不允许保存。
- 输入和自身一样的名称 → 正常保存，不算重复。

## TC-05 选择数据类型

- 单元格内容以select框显示。
- 下拉框只有 BOOL 和 INT 两个选项。
- 选 BOOL → 默认值自动变 TRUE。
- 选 INT → 默认值自动变 0。
- 切换类型时默认值跟着自动更新。

## TC-06 编辑 BOOL 默认值

- 单元格内容以input框显示。
- 输入 true/false/TRUE/FALSE 等各种大小写组合 → 统一显示为 TRUE 或 FALSE。
- 输入 yes、1 等非法值 → 报错。

## TC-07 编辑 INT 默认值

- 单元格内容以input框显示。
- 输入正常整数（正数、负数、零）→ 保存成功。
- 输入边界值 2147483647 和 -2147483648 → 保存成功。
- 输入超出范围、小数、字母、带空格、科学计数法等 → 报错。

## TC-08 编辑注释

- 单元格内容以input框显示。
- 输入任意文本都能保存。
- 清空也允许，不会报错。

## TC-09 综合场景

- 按需求中的故事卡片完整建出 3 行数据（Start/Stop/Count），验证数据正确。
- 删一行、改一行、再验证，确保增删改联动无误。
- 类型切换后再编辑默认值，校验联动正常。
- 添加多行再逐行删除至空，索引始终正确。

## TC-10 边界与异常

- 名称带空格时的处理。
- 一次性添加 100 行，页面不卡。
- 快速连续编辑不同字段，数据不丢。
- 刷新页面后数据是否还在（取决于是否做了持久化）。
